# 📇 Contact Application

A simple Python contact management app.

## Features
- 📝 Add contacts
- 📄 Display contacts
- 🔍 Search contacts

## Usage
1. Clone the repo:
    ```bash
    git clone https://github.com/yourusername/contact-application.git
    ```
2. Navigate to the directory:
    ```bash
    cd contact-application
    ```
3. Run the app:
    ```bash
    python contact_app.py
    ```
